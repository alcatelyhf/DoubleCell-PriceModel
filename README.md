模型功能说明
这个价格体系测算模型基于您提供的PDF文件中的"竞争性锚定，梯度韧性赋能"五步法架构设计，具有以下核心功能：

1. 单品测算
完整实现五步法架构：锚定MSRP → 设定代理出厂价 → 设计渠道利润体系 → 建立动态调控机制 → 结果分析

可调整所有关键参数：竞品价、策略类型、MSRP比例、出厂折扣、市场保护价比例等

实时计算并显示各级价格、利润空间和营收预测

可视化展示渠道利润分配

2. 多品测算
支持批量添加和管理多个产品

自动计算每个产品的价格体系

提供多产品汇总分析，包括平均MSRP比例、平均出厂折扣、预估总营收等

3. 策略场景模拟
提供四种预设策略：激进抢占市场、平衡发展、高端价值定位、渠道驱动型

每种策略自动设定相应的参数组合

直观比较不同策略下的价格体系差异

4. 数据导出功能
可将测算数据导出为CSV文件，可直接在Excel中打开和使用

包含单品测算、多产品测算和策略场景数据

5. 交互式体验
响应式设计，支持各种屏幕尺寸

实时计算，参数调整立即反映结果

清晰的公式说明和计算逻辑展示

计算逻辑核心
模型严格按照PDF中的五步法设计：

MSRP = 竞品价 × MSRP定价比例

代理赋能出厂价 = MSRP × 出厂价折扣

渠道利润空间 = MSRP - 代理赋能出厂价

最低市场保护价 = MSRP × 最低保护价比例

核心代理实际毛利率 = (渠道利润空间 + 返点金额) / MSRP
